# E-Commerce Marketplace Analysis – Power BI Dashboard

## Project Overview
This project analyzes an E-Commerce marketplace dataset to understand sales performance, customer behavior, and product profitability.

The goal is to transform raw transactional data into actionable business insights using interactive dashboards.

## Tools Used
- Power BI
- Data Cleaning
- Data Modeling
- Data Visualization

## Key KPIs
- Total Revenue
- Total Profit
- Total Quantity Sold
- Active Customers
- Repeat Customers
- Sales Trends Over Time

## Dashboard Insights
- Revenue and profit trends across multiple years
- Product and category performance
- Regional sales comparison
- Customer purchase behavior
- Shipping mode performance

## Project Outcome
The dashboard provides a clear overview of marketplace performance and helps identify opportunities to improve revenue, customer retention, and profitability.
